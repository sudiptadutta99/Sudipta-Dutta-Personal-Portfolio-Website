# Sudipta Dutta - Personal Portfolio Website

Welcome to my personal portfolio website! This project is a reflection of my journey as an aspiring software developer with a focus on Cloud Computing, Data Science, and Full Stack Development. It serves as an interactive platform where I can showcase my skills, share my projects, and connect with potential employers or collaborators.

## Features

- **Responsive Design:** The website is fully responsive, ensuring it looks great on any device, from desktop computers to mobile phones.
- **About Me Section:** Here, I introduce myself, share my interests, and outline my technical skills.
- **Skills:** This section lists my technical proficiencies, including programming languages, tools, and technologies I work with.
- **Experience:** I share details of my internship and other relevant professional experiences.
- **Education:** My academic background, including my B.Tech degree in Computer Science and Engineering, is highlighted here.
- **Services:** An overview of the services I offer, such as Web Development, Cloud Computing, and App Development.
- **Portfolio:** This section showcases some of the projects I've worked on, with links to their GitHub repositories.
- **Contact:** My contact information is provided along with a form for visitors to reach out to me.

## Technologies Used

- **HTML5**: Structure of the web pages.
- **CSS3**: Styling and layout of the website.
- **JavaScript**: Interactive elements and form submission handling.
- **Bootstrap**: Responsive grid system and components.
- **Font Awesome**: Icons used throughout the website.
